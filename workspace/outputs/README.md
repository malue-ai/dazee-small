# 销售数据可视化分析系统

## 功能特性

### 1. 数据生成
- ✅ 随机生成销售数据（可调整数量、日期范围、随机种子）
- ✅ 支持多产品类别（电子产品、服装、食品、家居、图书）
- ✅ 包含季节性趋势模拟

### 2. 交互式图表（6种类型）
- 📈 **时间序列折线图** - 展示数据随时间的变化趋势
- 📊 **类别对比柱状图** - 对比不同产品类别的表现
- 🥧 **地区分布饼图** - 分析各地区的销售占比
- 🔬 **散点相关性分析** - 探索变量之间的关系
- 📦 **箱线图分布** - 查看数据的分布和离群值
- 🌡️ **热力地图** - 产品类别与地区的交叉分析

### 3. 统计指标
- 💰 总销售额、平均订单数、客户满意度
- 📊 详细统计（总和、平均值、中位数、最大值、最小值、标准差）
- 📈 类别销售占比
- 📉 相关系数计算

### 4. 数据操作
- 🔍 按产品类别和地区筛选数据
- 📑 数据表格预览（可排序）
- 💾 导出筛选后的数据为 CSV 文件

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行应用
```bash
streamlit run app.py
```

### 访问应用
在浏览器中打开：http://localhost:8501

## 使用指南

1. **调整参数** - 在左侧边栏设置数据点数量、日期范围、随机种子
2. **选择类别** - 选择要分析的产品类别
3. **切换图表** - 从下拉菜单选择不同的可视化类型
4. **交互操作** - 使用单选按钮、滑块等控件调整显示内容
5. **筛选数据** - 在右侧统计面板筛选特定类别和地区
6. **导出数据** - 点击下载按钮保存筛选后的数据

## 技术栈
- Streamlit - Web应用框架
- Pandas - 数据处理
- NumPy - 数值计算
- Plotly - 交互式可视化
