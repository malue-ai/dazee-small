## 🎨 设计亮点

### 色彩系统
```
主色调：
- Primary: #667eea (紫色)
- Secondary: #764ba2 (深紫)
- Accent: #f093fb (粉色)

功能色：
- Success: #10b981 (绿色)
- Warning: #f59e0b (橙色)
- Danger: #ef4444 (红色)
```

### 布局方案
- Grid布局：路径卡片、课程卡片
- Flexbox布局：导航栏、卡片内容
- 响应式断点：768px、1280px

### 动画效果
- 浮动动画：3s ease-in-out
- 悬浮效果：transform translateY(-8px)
- 过渡时间：0.3s cubic-bezier
