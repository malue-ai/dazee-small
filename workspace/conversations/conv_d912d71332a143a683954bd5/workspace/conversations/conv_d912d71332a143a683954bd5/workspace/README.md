# 🎓 AI学习平台 - 使用文档

## 📖 项目简介

这是一个现代化的AI学习平台，为学习者提供完整的AI学习路径、丰富的课程资源、实战项目和智能学习助手。无论你是零基础的初学者，还是想要深入学习AI技术的开发者，这个平台都能为你提供帮助。

## ✨ 核心功能

### 1️⃣ 个性化学习路径
- **AI入门路径**：适合零基础学习者
- **机器学习路径**：掌握核心算法和应用
- **深度学习路径**：探索前沿技术
- **专项应用路径**：聚焦NLP、CV、AI Agent等特定领域

### 2️⃣ 丰富的课程体系
- 100+ 精品课程
- 多难度级别（入门/进阶/高级）
- 课程筛选和搜索功能
- 详细的课程信息展示

### 3️⃣ 实战项目练习
- 50+ 实战项目
- 从简单到复杂的进阶体系
- 包含手写数字识别、情感分析、智能对话机器人等热门项目
- 项目评分和完成统计

### 4️⃣ 学习进度追踪
- 可视化学习进度
- 学习天数和连续学习记录
- 课程完成统计
- 积分排名系统

### 5️⃣ AI智能助手
- 24/7在线答疑
- 课程推荐
- 学习建议
- 问题解答

### 6️⃣ 主题切换
- 亮色/深色主题自动切换
- 护眼模式
- 主题偏好记忆

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 建议屏幕分辨率：1920x1080 或更高

### 本地运行

1. **下载项目文件**
```bash
# 确保你已经拥有以下文件：
- index.html
- style.css
- script.js
```

2. **启动应用**

**方法一：直接打开**
- 双击 `index.html` 文件即可在浏览器中打开

**方法二：使用本地服务器（推荐）**
```bash
# 使用 Python 启动服务器
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

3. **访问应用**
- 打开浏览器访问：`http://localhost:8000`

## 📚 功能使用指南

### 导航栏
- **首页**：查看平台概览和统计数据
- **课程**：浏览和筛选课程
- **实践**：查看实战项目
- **进度**：查看个人学习进度

右上角功能：
- 🌙 主题切换按钮
- 🔐 登录按钮

### 学习路径选择

1. 浏览四种学习路径卡片
2. 点击感兴趣的路径查看详细信息
3. 点击"开始学习"按钮加入学习路径

### 课程学习

**筛选课程：**
- 点击顶部筛选按钮：全部 / 入门 / 进阶 / 高级
- 系统自动显示对应难度的课程

**查看课程详情：**
- 每个课程卡片包含：
  - 课程封面
  - 课程标题和描述
  - 讲师信息
  - 学习时长
  - 学生数量
  - 课程评分
  - 技术标签

**开始学习：**
- 点击"立即学习"按钮加入课程

### 实战项目

**项目信息：**
- 难度等级（入门/进阶/高级）
- 预计完成时间
- 项目描述
- 技术栈标签
- 完成人数和评分

**开始实践：**
- 点击"开始实践"按钮启动项目

### 学习进度

**查看统计：**
- 学习天数和连续学习记录
- 完成课程数和证书数量
- 实战项目完成情况
- 学习积分和排名

**最近学习：**
- 查看最近的学习活动
- 时间线展示
- 完成状态标记

### AI助手使用

**打开助手：**
- 点击右下角的机器人图标

**功能：**
1. **推荐课程**：点击建议按钮或输入"推荐课程"
2. **解答疑问**：输入你的问题
3. **学习建议**：获取学习方法和技巧

**快捷建议：**
- 推荐课程
- 解答疑问
- 学习建议

## 🎨 主题切换

### 切换方式
- 点击右上角的月亮/太阳图标
- 系统自动保存你的主题偏好

### 主题特点
- **亮色主题**：清新明亮，适合白天使用
- **深色主题**：护眼舒适，适合夜间使用
