## 🎯 三步启动

### 方法一：直接打开（最简单）

1. **找到项目文件夹**
2. **双击 `index.html`**
3. **开始使用！** ✨

### 方法二：本地服务器（推荐）

```bash
# 进入项目目录
cd /path/to/ai-learning-platform

# Python 启动服务器
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000

# 打开浏览器访问
# http://localhost:8000
```

## 🎮 5分钟体验教程

### Step 1: 浏览学习路径 (1分钟)
1. 打开应用后向下滚动
2. 查看4种学习路径
3. 点击感兴趣的路径卡片

### Step 2: 筛选课程 (1分钟)
1. 点击顶部导航"课程"
2. 使用筛选按钮：全部/入门/进阶/高级
3. 浏览课程卡片
4. 点击"立即学习"按钮

### Step 3: 查看实战项目 (1分钟)
1. 点击顶部导航"实践"
2. 浏览不同难度的项目
3. 查看项目标签和评分

### Step 4: 体验AI助手 (1分钟)
1. 点击右下角机器人图标
2. 点击快捷建议按钮
3. 或输入你的问题

### Step 5: 切换主题 (1分钟)
1. 点击右上角月亮/太阳图标
2. 体验深色/亮色主题

## ⚠️ 注意事项

1. **浏览器兼容性**
   - 使用Chrome、Firefox、Safari、Edge等现代浏览器
   - IE浏览器不支持

2. **性能建议**
   - 建议屏幕分辨率：1920x1080 或更高
   - 清除浏览器缓存以获得最佳体验

3. **数据存储**
   - 当前使用本地存储
   - 清除浏览器数据会丢失进度
