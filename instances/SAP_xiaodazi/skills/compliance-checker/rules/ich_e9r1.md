# ICH E9(R1) 合规规则（5 条）
E9R1-01 [CRITICAL]: Estimand 必须包含五属性（Population/Endpoint/Treatment/ICE/Summary）
E9R1-02 [MAJOR]: ICE 处理策略必须为标准类型（treatment policy/composite/hypothetical/principal stratum/while on treatment）
E9R1-03 [MAJOR]: Population-level summary 与分析方法一致
E9R1-04 [CRITICAL]: SAP Estimand 与 Protocol 一致
E9R1-05 [MINOR]: 使用 ICH E9(R1) 标准术语
