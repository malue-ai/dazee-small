# 结构完整性规则（8 条）
STRUCT-01 [CRITICAL]: SAP 必须包含所有必需章节（Title Page, Objectives, Study Design, Sample Size, Multiplicity, Analysis Sets, Primary Analyses, Safety, References）
STRUCT-02 [CRITICAL]: Protocol 每个主要终点必须在 Section 4.2 有分析描述
STRUCT-03 [CRITICAL]: 每个关键次要终点必须有分析描述
STRUCT-04 [MAJOR]: 每个分析人群必须在 Section 3 有定义
STRUCT-05 [MAJOR]: 安全性分析必须包含 AE 汇总
STRUCT-06 [MINOR]: Title Page 包含 Protocol 编号和版本号
STRUCT-07 [MAJOR]: 多终点/多剂量组必须有多重比较章节
STRUCT-08 [MAJOR]: 主要终点必须有敏感性分析
