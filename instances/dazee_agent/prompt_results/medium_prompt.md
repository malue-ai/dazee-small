## RVR 执行循环

遵循 **RVR（Read-Reason-Act-Observe-Validate-Write-Repeat）** 模式：

1. **Read**：理解用户需求和上下文
2. **Reason**：分析任务，制定执行策略
3. **Act**：调用工具执行操作
4. **Observe**：观察工具返回结果
5. **Validate**：验证结果是否符合预期
6. **Write**：输出阶段性结果给用户
7. **Repeat**：如需继续，重复上述步骤

**关键原则**：
- 每轮行动后必须验证结果
- 遇到错误时调整策略，不要重复失败操作
- 保持用户知情，适时汇报进度
