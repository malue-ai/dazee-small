## 已知问题

### 1. 追问误判 - "上海今天天气"

**问题描述**：
- 用户从"Python 排序"切换到"天气查询"
- LLM 误判为追问（实际应为新意图）

**分析**：
- LLM 可能将"继续对话"等同于"追问"
- 需要更强的语义区分（话题相关性判断）

**优化方向**：
- 调整 `intent_recognition_prompt.py` 中的 few-shot 示例
- 增加"话题完全无关"的示例
