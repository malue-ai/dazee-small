# Dazee 7大核心能力 Use Cases（阶段一 1.29打榜版本）

本文档基于 Dazee 产品定义的 7 个核心能力，从**用户需求视角**提供典型 use cases。每个 use case 包括用户的自然语言输入和期望获得的输出结果。

---

## 1. 更懂你，更贴心

### 能力描述
Dazee 拥有长效记忆，能在对话中自动延续上下文，无需你重复说明。它能主动学习您的偏好与习惯，提供个性化提醒与温馨建议。越聊，Dazee 越懂你~

### Use Case 1.1: 延续上下文对话

**用户第一轮对话:**
```
帮我分析一下我们公司去年的销售数据，重点看华东区域。
```

**Dazee 输出:**
- 分析华东区域销售数据
- 返回：华东区 2023 年销售额 2500 万，同比增长 18%

**用户第二轮对话（几天后）:**
```
那华北区呢？
```

**期望输出:**
- Dazee 自动理解"华北区"指的是销售数据分析
- 无需重复说明"分析销售数据"
- 返回：华北区 2023 年销售额 1800 万，同比增长 12%
- 主动对比：华北区比华东区低 700 万

### Use Case 1.2: 个性化偏好记忆

**用户对话历史:**
- 第 1 次：生成 PPT 时选择了"商务风格"
- 第 2 次：生成 PPT 时选择了"商务风格"
- 第 3 次：生成报告时要求"数据来源必须标注"

**用户新请求:**
```
帮我做一个产品介绍 PPT。
```

**期望输出:**
- Dazee 主动提示："根据您的习惯，我会使用商务风格，并在数据部分标注来源，可以吗？"
- 或者直接应用用户偏好，生成商务风格的 PPT
- 减少重复沟通成本

### Use Case 1.3: 主动提醒与建议

**场景背景:**
- 用户每周一早上都会问"本周有什么重要安排"
- 已连续 3 周

**用户周一早上打开 Dazee:**

**期望输出:**
- Dazee 主动问候："早上好！我注意到您通常会在周一查看本周安排，这是为您准备的本周日程摘要：..."
- 无需用户主动询问
- 显示：本周 3 个会议、2 个截止日期、1 个待跟进客户

## 2. 提供专业写作支持

### 能力描述
告别从零开始的写作压力！无论是商务邮件、营销文案还是个人简历，Dazee 能提供覆盖全场景的智能写作支持，为您快速生成高质量初稿，显著提升内容产出效率。

### Use Case 2.1: 商务邮件撰写

**用户需求:**
```
帮我写一封邮件给李总，说明我们这个月的项目进度有点延期，需要申请延期到下个月 15 号，态度要诚恳但专业。
```

**期望输出:**
- 生成完整邮件：
  - 主题：关于 XX 项目进度调整的申请
  - 称呼：尊敬的李总
  - 正文：
    - 简要汇报当前进度（已完成 80%）
    - 说明延期原因（需求变更、资源调配）
    - 提出延期申请（延至下月 15 日）
    - 保证质量承诺
    - 表达歉意和感谢
  - 落款：XXX 敬上
- 语气专业、诚恳，逻辑清晰

### Use Case 2.2: 营销文案生成

**用户需求:**
```
我们要推广一款智能手表，目标用户是运动爱好者，帮我写一段小红书种草文案，要口语化、有代入感，200 字左右。
```

**期望输出:**
- 小红书风格文案：
  - 开头：用痛点引入（"跑步时手机太重？"）
  - 中间：产品卖点（心率监测、GPS 轨迹、续航 7 天）
  - 结尾：行动号召（"评论区抽 3 人送运动手环！"）
  - 配 emoji 和话题标签 #运动手表 #跑步装备
- 总字数 180-220 字
- 口语化、有画面感

### Use Case 2.3: 个人简历优化

**用户需求:**
```
我要应聘产品经理，工作经历是在电商公司做过 2 年运营，帮我把这段经历写得更专业，突出我的数据分析能力和项目管理经验。
```

**期望输出:**
- 优化后的工作经历描述：
  - 标题：XX 电商平台 - 高级运营专员（2021-2023）
  - 核心职责：
    - 负责 XX 品类运营，管理 50+ SKU，年销售额 800 万
    - 通过数据分析优化转化率，使 GMV 提升 25%
    - 主导 3 次大型促销活动，累计 UV 超 100 万
  - 使用产品经理相关词汇（用户洞察、需求分析、迭代优化）
  - 量化成果（数字、百分比）

## 3. 一键生成深度报告

### 能力描述
告别繁琐！仅需输入关键词，Dazee 可自动完成行业扫描、竞品分析与趋势研判，并输出结构清晰、数据可溯源的专业报告，为你提供科学的决策依据。

### Use Case 3.1: 行业趋势报告

**用户需求:**
```
我想了解"AI 教育"这个赛道的发展趋势，生成一份行业分析报告。
```

**期望输出:**
- 自动生成 15-20 页报告（Word/PDF）：
  - **第一章：行业概况**
    - 市场规模：2024 年全球 AI 教育市场 XX 亿美元
    - 增长趋势：年复合增长率 28%
    - 政策环境：教育部 XX 文件支持
  - **第二章：竞品分析**
    - 主要玩家：好未来、作业帮、猿辅导等
    - 产品对比表格（功能、定价、用户量）
  - **第三章：技术趋势**
    - AI 自适应学习、虚拟助教、知识图谱
  - **第四章：未来展望**
    - 预测 2025-2027 年发展方向
  - **附录：数据来源**
    - 所有数据标注来源（艾瑞咨询、36氪等）

### Use Case 3.2: 竞品深度分析

**用户需求:**
```
帮我分析一下"钉钉"和"飞书"这两款协作工具的差异，我要决定公司用哪个。
```

**期望输出:**
- 生成竞品对比报告：
  - **核心功能对比表**（通讯、文档、项目管理、OKR、审批）
  - **定价对比**（免费版功能、付费版价格）
  - **用户评价**（从应用商店抓取评分和高频评论）
  - **优缺点总结**
    - 钉钉：审批流强、阿里生态、界面传统
    - 飞书：协作体验好、文档强、价格略高
  - **推荐建议**
    - 如果重视审批流和考勤 → 钉钉
    - 如果重视协作体验和文档 → 飞书
- 数据来源标注（官网、用户评价、第三方报告）

### Use Case 3.3: 市场调研报告

**用户需求:**
```
我们计划进入"预制菜"市场，帮我生成一份市场调研报告，包括市场规模、目标客户、竞争格局。
```

**期望输出:**
- 自动生成市场调研报告：
  - **市场规模**：2024 年中国预制菜市场 4200 亿，预计 2027 年破 1 万亿
  - **目标客户画像**
    - C 端：25-40 岁上班族，注重便捷和健康
    - B 端：餐饮连锁、团餐企业
  - **竞争格局**
    - 头部品牌：珍味小梅园、味知香
    - 市场集中度低（CR5 < 15%）
  - **进入建议**
    - 建议切入细分品类（如酸菜鱼、佛跳墙）
    - 渠道建议：线上（盒马、叮咚）+ 线下（商超）
  - **风险提示**：冷链成本高、食品安全要求严

## 4. 多文件数据全自动分析

### 能力描述
降低数据分析门槛！Dazee 支持表格、文档、图片等多种文件同时上传，自动完成数据清洗、建模与可视化分析，能直接输出可用的结论与策略建议，让数据真正驱动业务决策。

### Use Case 4.1: 销售数据多维分析

**用户操作:**
- 上传 3 个文件：
  - `2024年Q1销售数据.xlsx`（各区域销售明细）
  - `2024年Q2销售数据.xlsx`
  - `客户反馈汇总.docx`（文字描述）

**用户需求:**
```
帮我分析一下上半年的销售情况，找出表现最好的区域和产品，结合客户反馈给我一些改进建议。
```

**期望输出:**
- **数据自动清洗**（Dazee 后台处理，用户无感知）
- **分析结果：**
  - 上半年总销售额：5800 万，同比增长 22%
  - 最佳区域：华南区（占比 32%，增长 35%）
  - 最佳产品：A 系列（销量增长 50%）
  - 客户反馈关键词：价格合理（45次提及）、配送慢（23次提及）
- **可视化图表：**
  - 区域销售对比柱状图
  - 产品销量占比饼图
  - 月度增长趋势线图
- **策略建议：**
  - 建议加大华南区市场投入
  - 优化配送速度（客户痛点）
  - A 系列可考虑提价或推高端版本

### Use Case 4.2: 财务数据对比分析

**用户操作:**
- 上传 2 个文件：
  - `2023年财务报表.xlsx`
  - `2024年财务报表.xlsx`

**用户需求:**
```
对比一下这两年的财务情况，看看哪些科目变化比较大。
```

**期望输出:**
- **同比变化表格：**
  - 营收：2023 年 8000 万 → 2024 年 9500 万（+18.75%）
  - 成本：2023 年 5000 万 → 2024 年 6200 万（+24%）
  - 净利润：2023 年 1200 万 → 2024 年 1100 万（-8.3%）
- **关键发现：**
  - ⚠️ 成本增速（24%）超过营收增速（18.75%），利润率下降
  - 人力成本增加 30%（主要变化项）
  - 营销费用增加 15%
- **可视化：**
  - 收入/成本/利润对比柱状图
  - 费用结构变化桑基图
- **建议：**
  - 关注成本管控，特别是人力成本优化
  - 评估营销 ROI，提升投放效率

### Use Case 4.3: 混合数据综合分析

**用户操作:**
- 上传 4 个文件：
  - `用户调研问卷结果.xlsx`（1000 条问卷数据）
  - `产品使用日志.csv`（10 万条行为数据）
  - `竞品功能对比.png`（截图）
  - `产品需求文档.docx`

**用户需求:**
```
综合这些资料，帮我分析一下我们产品的优缺点，以及下个版本应该重点改进什么。
```

**期望输出:**
- **综合分析报告：**
  - **优点：**
    - 界面美观（问卷满意度 4.2/5）
    - 核心功能稳定（日志崩溃率 < 0.1%）
  - **缺点：**
    - 缺少批量操作（问卷提及 78 次）
    - 加载速度慢（日志平均响应 2.3 秒，行业标准 < 1 秒）
    - 竞品对比：缺少 AI 推荐功能（对手已支持）
  - **下版本改进建议（优先级排序）：**
    1. P0：优化加载速度（影响 80% 用户）
    2. P0：新增批量操作（高频需求）
    3. P1：探索 AI 推荐功能（对齐竞品）
- **图表：**
  - 问卷满意度雷达图
  - 功能使用频次热力图
  - 竞品功能对比矩阵

## 5. 零代码构建轻量应用

### 能力描述
不用懂编程，Dazee 通过需求描述，即可为您生成专属网页应用。从项目管理看板到个人工具，Dazee 让灵感快速变为现实。

### Use Case 5.1: 项目管理看板

**用户需求:**
```
我们团队需要一个简单的项目看板，能添加任务、分配给成员、标记状态（待开始、进行中、已完成），不需要太复杂。
```

**期望输出:**
- 生成一个网页应用（可直接访问的 URL）
- **功能：**
  - 看板视图（三列：待开始、进行中、已完成）
  - 点击"新建任务"弹出表单：任务名称、负责人、截止日期、优先级
  - 拖拽任务卡片切换状态
  - 任务卡片显示：标题、负责人头像、截止日期、优先级标签
  - 支持按负责人筛选
- **交互体验：**
  - 界面清爽，类似 Trello
  - 移动端适配（手机也能用）
- **数据存储：**
  - 数据保存在云端，团队成员可协同访问

### Use Case 5.2: 客户线索记录表

**用户需求:**
```
我是销售，需要一个工具记录客户线索，包括客户姓名、公司、联系方式、意向等级、跟进记录，能导出 Excel。
```

**期望输出:**
- 生成客户线索管理应用
- **功能：**
  - 客户列表（表格展示）：姓名、公司、手机、意向等级、最后跟进时间
  - 点击"添加客户"填写表单
  - 点击客户进入详情页：
    - 基本信息
    - 跟进记录时间线（每次跟进的时间、内容）
    - 添加跟进记录（文本框 + 保存）
  - 意向等级标签：高（红色）、中（黄色）、低（灰色）
  - 搜索功能（按姓名/公司）
  - 导出为 Excel 按钮
- **额外特性：**
  - 7 天未跟进的客户自动标红提醒

### Use Case 5.3: 个人时间记录工具

**用户需求:**
```
我想做一个时间记录工具，每天记录我在不同项目上花了多少时间，能看到周报和月报统计。
```

**期望输出:**
- 生成时间记录应用
- **功能：**
  - **记录页面：**
    - 选择项目（下拉菜单：工作、学习、健身、娱乐）
    - 输入时长（小时/分钟）
    - 备注（可选）
    - 日期选择器（默认今天）
  - **统计页面：**
    - 周视图：本周各项目时长柱状图
    - 月视图：本月各项目时长饼图
    - 显示总时长和占比
  - **历史记录：**
    - 按日期倒序展示所有记录
    - 可编辑或删除
- **视觉：**
  - 配色温馨（浅色系）
  - 图表清晰易读

## 6. 自由搭建个性化工作台

### 能力描述
减少应用之间的切换，提升工作流程！Dazee 支持用户像拼装乐高一样，自由搭建个性化工作台，自定义功能和工作流，打造一站式、贴合自身思维习惯的效率中心。

### Use Case 6.1: 销售工作台

**用户需求:**
```
我是销售经理，每天要查看销售数据、跟进客户、发邮件、查竞品信息，能帮我搭建一个工作台，把这些功能集成在一起吗？
```

**期望输出:**
- 生成个性化销售工作台（单一页面）
- **布局：**
  - 左上：今日数据卡片（今日销售额、新增线索、待跟进客户数）
  - 左下：客户列表（高优先级客户置顶）
  - 右上：快捷操作区
    - "发邮件"按钮（点击打开邮件编辑器）
    - "查竞品"按钮（调用搜索功能）
    - "生成周报"按钮
  - 右下：最近跟进记录（时间线）
- **交互：**
  - 点击客户 → 弹出详情和跟进表单
  - 数据卡片可点击 → 跳转详细报表
  - 支持拖拽调整模块位置和大小

### Use Case 6.2: 内容创作者工作台

**用户需求:**
```
我是自媒体运营，每天要写文章、找素材、管理选题、追踪数据，帮我搭建一个内容创作工作台。
```

**期望输出:**
- 生成内容创作工作台
- **模块组成：**
  - **选题池**（左侧）
    - 显示所有选题（待写、写作中、已发布）
    - 点击选题 → 进入写作器
  - **素材库**（右侧）
    - 最近保存的图片、文章链接
    - 支持拖拽到写作器
  - **数据看板**（顶部横幅）
    - 本周发布文章数、阅读量、涨粉数
  - **快捷工具栏**（悬浮）
    - "搜索热点"、"生成大纲"、"找配图"
- **自定义能力：**
  - 用户可以添加/删除模块
  - 调整布局（拖拽排列）
  - 保存为"我的工作台"

### Use Case 6.3: 研发项目管理工作台

**用户需求:**
```
我是技术 Leader，需要一个工作台看到所有项目的进度、成员状态、代码提交情况、线上bug数量。
```

**期望输出:**
- 生成研发管理工作台
- **模块组成：**
  - **项目总览**（顶部）
    - 各项目进度条（百分比）
    - 红色预警：延期项目
  - **团队状态**（左侧）
    - 成员列表 + 当前任务
    - 工作负载：空闲/正常/饱和（绿/黄/红）
  - **代码动态**（右上）
    - 最近 24 小时代码提交次数
    - 代码审查待处理数
  - **线上监控**（右下）
    - 今日 Bug 数量
    - P0/P1/P2 分级显示
    - 点击 → 跳转 Bug 详情
- **集成能力：**
  - 对接 GitLab/GitHub API（代码数据）
  - 对接项目管理系统（进度数据）
  - 对接监控平台（Bug 数据）

## 7. 轻松生成设计级PPT

### 能力描述
Dazee 覆盖 20+ 应用场景，可根据主题自动优化视觉呈现，让您专注于内容本身，高效输出具有专业设计感的演示文稿。

### Use Case 7.1: 产品发布会 PPT

**用户需求:**
```
我们要开新品发布会，产品是"智能音箱 Pro"，帮我做一个发布会 PPT，要有科技感，包括产品外观、核心功能、价格、上市时间。
```

**期望输出:**
- 生成 15-20 页设计级 PPT
- **内容结构：**
  - 封面：产品名称 + 主视觉（科技风背景）
  - 开场：市场痛点（"传统音箱无法理解复杂指令"）
  - 产品介绍：外观图 + 3 大卖点（AI 语音、全屋控制、音质升级）
  - 功能展示：每个功能 1-2 页（配图/视频占位符）
  - 技术参数：表格展示（处理器、内存、续航等）
  - 价格政策：大字号突出价格，配"限时优惠"标签
  - 上市时间：时间线图（预售、开卖、铺货）
  - 结尾：品牌 slogan + 联系方式
- **设计风格：**
  - 科技蓝配色
  - 大图排版（少文字）
  - 动画效果：进场渐显
- **交付物：**
  - PPTX 文件（可编辑）
  - 演讲者备注（每页关键话术）

### Use Case 7.2: 季度总结汇报 PPT

**用户需求:**
```
我要向老板汇报 Q1 的工作成果，包括完成的项目、数据成绩、遇到的问题、下季度计划，风格要商务专业。
```

**期望输出:**
- 生成 12-15 页商务风 PPT
- **内容结构：**
  - 封面：Q1 工作总结 + 部门名称
  - 目录：4 个章节
  - **第一章：核心数据**
    - 大数字展示（销售额、用户量、项目数）
    - 同比/环比增长百分比（绿色上升箭头）
  - **第二章：重点项目**
    - 3 个项目卡片（项目名、成果、亮点）
    - 配时间线图
  - **第三章：问题与挑战**
    - 列表展示（3-4 条）
    - 每条问题 + 已采取的措施
  - **第四章：Q2 计划**
    - 甘特图或表格（任务、负责人、时间）
  - 结尾：谢谢聆听
- **设计风格：**
  - 深蓝/灰色商务配色
  - 图表为主（柱状图、折线图、饼图）
  - 排版规整，留白舒适

### Use Case 7.3: 培训课程 PPT

**用户需求:**
```
我要做一个"时间管理技巧"的内部培训，1 小时课程，需要 PPT 包括理论介绍、方法演示、实战案例，风格要轻松易懂。
```

**期望输出:**
- 生成 30-40 页培训 PPT
- **内容结构：**
  - 封面：课程标题 + 讲师信息
  - 破冰：互动问题（"你每天有多少时间被浪费？"）
  - **理论部分（8 页）：**
    - 什么是时间管理
    - 常见误区
    - 四象限法则（配图解）
  - **方法部分（15 页）：**
    - 番茄工作法（步骤图）
    - GTD 任务管理（流程图）
    - 每日计划模板（表格）
  - **案例部分（5 页）：**
    - 真实案例：某员工如何从"忙乱"到"高效"
    - 前后对比（混乱日程 vs 优化日程）
  - **实战练习（3 页）：**
    - 现场任务：规划明天的日程
    - 提供练习表格
  - **总结（2 页）：**
    - 核心要点回顾
    - 推荐工具/书籍
- **设计风格：**
  - 明亮配色（橙黄/浅蓝）
  - 插画风图标
  - 大字标题，易读正文
  - 互动提示（如"请大家思考 1 分钟"）

## Use Case 使用建议

### 测试验证
每个 use case 可作为：
- **功能验收标准**：实际执行是否达到期望输出
- **性能基准**：响应时间、准确率
- **用户体验评估**：交互是否流畅、结果是否满足需求

### 迭代优化
基于用户反馈，持续优化：
- 输出质量（准确性、完整性）
- 响应速度
- 个性化程度

### 场景扩展
7 大能力可组合使用，例如：
- **能力 3 + 能力 7**：生成深度报告 → 转化为 PPT 汇报
- **能力 4 + 能力 2**：分析数据 → 撰写数据分析邮件
- **能力 5 + 能力 6**：构建应用 → 集成到个人工作台
